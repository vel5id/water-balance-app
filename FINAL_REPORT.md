## 💡 Ключевые возможности

### 1️⃣ Три метода в один клик
- **Theil-Sen:** Быстро, просто, стабильно ⚡
- **SARIMA:** Авторегрессионный, экспериментальный 🔬
- **Prophet:** Продвинутый, с интервалами 🎯

### 2️⃣ Полная обработка ошибок
```python
Если Prophet не установлена:
  ✓ Показать предупреждение
  ✓ Fallback на Theil-Sen
  ✓ Продолжить симуляцию

Если SARIMA требует память:
  ✓ Ограничить историю (max_history=730)
  ✓ Показать информацию
  ✓ Продолжить

Если SARIMA падает (сезонная):
  ✓ Автоматически попробовать non-seasonal
  ✓ Показать какой использовался
  ✓ Вернуть результат
```

### 3️⃣ Информативный вывод
```
✅ SARIMA (ARIMA): AIC=1234.56, BIC=1289.45
✅ Prophet: 1825 observations used, yearly+weekly seasonality
⚠️ SARIMA failed: OOM. Falling back to Theil-Sen
ℹ️ Prophet requires: pip install pystan prophet
```

### 4️⃣ Гибкие параметры
```python
# Каждый параметр настраивается:
build_prophet_forecast(
    series,
    yearly_seasonality=True,      ← on/off
    weekly_seasonality=True,      ← on/off
    interval_width=0.95           ← 5-95%
)

build_sarima_forecast_enhanced(
    series,
    max_history=730,              ← память
    max_p=7, max_q=7,             ← сложность
    seasonal=True                 ← on/off
)
```

### Тест в Python
```python
from wbm.forecast import *

# Данные
series = pd.Series([...], index=pd.date_range(...))

# Протестировать все три
ts_result = build_robust_season_trend_series(series)
sarima_forecast, _ = build_sarima_forecast_enhanced(series)
prophet_forecast, _ = build_prophet_forecast(series)

# Сравнить
print(f"Theil-Sen:  {len(ts_result.deterministic)}")
print(f"SARIMA:     {len(sarima_forecast)}")
print(f"Prophet:    {len(prophet_forecast)}")
```
